# Express React Graphql App

#### Backend: Node (Express in Typescript)
#### CI/CD : Travis

<br>
Checklist

- Install Package Dependencies
- Build Api from source



### Install Package Dependencies
```
$ cd api
$ yarn install
```

### Build App
```
$ cd api
$ yarn build
```

### Start Api (Dev Mode)
```
$ cd api
$ yarn watch-ts
$ yarn watch-js
```

### Start Api (Prod Mode)
```
$ cd api
$ yarn start
```
