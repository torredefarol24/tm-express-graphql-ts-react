# Express React Graphql App

##### Backend: Node (Express in Typescript)
##### CI/CD : Travis


Checklist

- Install Package Dependencies
- Build App



### Install Package Dependencies
```
$ yarn install
```

### Build App
```
$ yarn build
```

### Start Api (Dev Mode)
```
$ yarn watch-ts
$ yarn watch-js
```

### Start Api (Prod Mode)
```
$ yarn start
```
